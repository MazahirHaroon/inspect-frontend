Inspect Frontend — Intro to JavaScript

Inspect Frontend is a casual tech hangout for devs to share what they’re learning, ask doubts, post job leads, and discuss JavaScript and frontend engineering. No spam. No pressure. Just curious minds and clean code. This is our Intro to JavaScript session, where we'll explore how code runs in the browser, step by step.

Quick note for students: Please read the topics and timetable before the first session. This page shows exactly what we'll cover and when — it helps you prepare and follow along with clarity. If anything is unclear, mention it on the call.
